# springboot-atomikos-mybatis-plus-jdbctemplate-mysql-postgresql
- spring boot 2.5.4
- mybatis-plus 3.4.3.3
- mysql 5.7
- postgresql 13
- sqlserver 2016
- atomikos
- 整合单数据库事务、多数据库事务。

关键就是把单数据库事务和多数据库事务分开，AtomikosDataSource效能比较低。
